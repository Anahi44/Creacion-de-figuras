# Proyecto: Generador de Figuras con Comandos de Voz

Este proyecto, desarrollado en Java, permite crear y manipular figuras geométricas en un panel interactivo.  
Incluye funcionalidades como:

- Creación de triángulos, cuadrados y rectángulos mediante botones y comandos de voz.
- Cambio de color y tamaño de las figuras.
- Agrupación y movimiento de figuras en el panel.
- Estadísticas sobre figuras creadas y eliminadas.
- Exploración inicial de control por cámara (en desarrollo).

## Tecnologías utilizadas
- Java
- Librerías de reconocimiento de voz
- Interfaz gráfica con Swing
