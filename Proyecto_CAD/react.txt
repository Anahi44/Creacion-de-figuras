instalaciones necesarias para el frontend 
1.- instalar los siguientes elementos
	https://nodejs.org/en/
	https://marketplace.visualstudio.com/items?itemName=burkeholland.simple-react-snippets
	https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets
	https://chromewebstore.google.com/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi?hl=es&authuser=1
	https://chromewebstore.google.com/detail/redux-devtools/lmhkpmbekcpmknklioeibfkpmmfibljd?hl=es
2.- Hacer correr el siguiente comando en la terminal:
	npm install react-router-dom
3.- Colocar el siguiente link en el documento de /principal/index.html
	<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">
4.- hacer correr el sistemas con:
	npm start
